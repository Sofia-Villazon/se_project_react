Making a Counter with Closure
Task4 / 4

The closure is ready, but we need to make sure it works. We’ll need to add some temporary logging in the createCounter() function for this purpose:

  • After incrementing the counter variable, log it to the console.
  • After the createCounter() declaration, create a variable myCounter and assign the createCounter() function’s return value.
  • Call this variable’s increaseCounter() method a few times and check the results in the console.

You should see that the value is increasing, starting with either a 0 or a 1, depending on the exact placement of your log statement.
